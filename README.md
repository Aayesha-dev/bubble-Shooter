# 🎈 Bubble Shooter Game

A classic **Bubble Shooter** game built with **C++**, **OpenGL**, and **Qt**.  
The game features **10 progressive levels**, **matching mechanics**, **score tracking**, a **level unlock system**, and a **bubble swap feature**!

---

## 🎮 Game Features

| Feature | Description |
|---------|-------------|
| **🔴 10 Colors** | 10 different colored bubbles to match and shoot |
| **💥 Match-3 Mechanic** | Shoot bubbles to form groups of 3 or more to pop them |
| **📈 10 Levels** | Each level increases in complexity and difficulty |
| **🔒 Level Lock System** | Only Level 1 is unlocked initially; complete a level to unlock the next |
| **🏠 Home Screen** | Navigate back to the home screen from any stage |
| **📊 Score Tracker** | View your current score during gameplay |
| **⏳ Loading Screen** | Smooth loading transitions between levels |
| **🎯 Increasing Difficulty** | Each level introduces new challenges and bubble arrangements |
| **👀 Next Bubble Preview** | See the next bubble that will appear in your shooter |
| **🔄 Bubble Exchange** | Swap the current bubble with the next one for better strategy |

---

##  How to Play

1. **Aim** – Move your mouse to aim the shooter
2. **Shoot** – Click to shoot the bubble
3. **Match** – Match 3 or more bubbles of the same color to pop them
4. **Preview** – Look at the next bubble to plan your move
5. **Exchange** – Swap the current bubble with the next one if needed
6. **Clear** – Clear all bubbles to complete the level
7. **Progress** – Unlock new levels as you complete each stage

---

##  Bubble Exchange System

| Feature | Description |
|---------|-------------|
| **Next Bubble Display** | Shows the upcoming bubble color on the screen |
| **Exchange Button** | press s button to swap your current bubble with the next one |
| **Strategic Play** | Use exchange to create better matches and clear levels faster |

## 🚀 How to Run

### Option 1: Using Qt Creator (Recommended)

1. Install **Qt** from [qt.io/download](https://www.qt.io/download)
2. Open **Qt Creator**
3. Go to **File → Open File or Project**
4. Select `bubble-shooter.pro` file
5. Click **Build** (Hammer icon) to compile
6. Click **Run** (Green Play icon) to start the game
